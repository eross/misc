How to use Vermont Recipes

Vermont Recipes doesn't do anything useful at all. It simply demonstrates how a variety of user controls and other application features work in a typical Mac OS X application.

The book on which the application is based, Cocoa Recipes for Mac OS X—The Vermont Recipes, Peachpit Press, www.peachpit.com, explains how to create all of these controls and features using the Objective-C programming language and Apple's Cocoa frameworks. To learn how to buy the book, choose Help > Read Me or Vermont Recipes > About Vermont Recipes.

When you launch Vermont Recipes or click its icon in the Dock while no document window is open, an untitled document window opens. Choose File > New to open additional untitled document windows. After saving and closing a document, reopen it by choosing File > Open or File > Open Recent or double-click the document's icon in the Finder.

Each Vermont Recipes document has a main window containing a tab view with three panes. Each pane contains a group of user controls of a similar kind: buttons, sliders, and text fields. Click a tab to go to that pane, or click the Back and Next buttons to cycle through all of them.

A document's main window also has a drawer containing a scrolling text view where you can type or paste notes. Click the Notes button in the main document window to open and close the drawer. To resize the drawer and the notes view horizontally, drag the drawer's outer edge left or right, and you can close the drawer by dragging its edge almost all the way back to the window. The notes view supports all standard Mac OS X editing features, including multiple undo and redo, drag-and-drop editing, and formatting using the commands in the Format menu.

The Open in Window button at the bottom of the drawer opens a second, ancillary document window. This window contains a split view allowing you to read and edit different parts of the notes from the drawer's notes view. The notes window can be resized and moved anywhere on the desktop.

Two kinds of help are available explaining how to use individual features of the application:

1. Help tags. Move the mouse over any user control in the application and pause there. You don't have to click. After a moment, a small Help tag appears briefly explaining what the control is or what it does. After a few more moments, the Help tag fades away.

2. Context help. Press the Help key on your keyboard or, if you're using a PowerBook or a keyboard without a Help key, choose Help > Turn Context Help On. The cursor turns into a large question mark. Click any user control. A large Help tag appears giving more detailed information about the control and its use. Click anywhere to dismiss the tag.
