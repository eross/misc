Welcome to img2icns help!
Img2icns is simple converter program from images to macosx icons.

Using img2icns: Usage is pretty simple, select the prefered output:
- .icns (iconset format, you wont see icon if macosx « 10.4)
- directory with icon (a directory will be created and icon applied on that, so you can copy/paste)

Then simply drop as many image you want on img2icns dock icon, icons will be created in same location of image.

How to change icon to a program/folder: Click on icon you want to use, press Cmd-i, then in the info panel click on upper left icon and copy it pressing Cmd-c. Same operation on program/folder you want to change icon, but instead of Cmd-c for coping after clicking on upper left icon use Cmd-v for paste new icon. Job done.
